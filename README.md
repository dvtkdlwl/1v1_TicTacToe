# 1v1 Tic Tac Toe (Local Multiplayer)


A simple terminal-based Tic Tac Toe game written in Python.
Two players take turns playing on the same keyboard.


This project is an early learning exercise and is uploaded for archival and progress-tracking purposes.


## How to Run


Make sure Python 3 is installed, then run:


```bash
python tictactoe.py
