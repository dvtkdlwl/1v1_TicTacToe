# Tic Tac Toe (Local Multiplayer)

A simple terminal-based Tic Tac Toe game written in Python.  
Two players take turns playing on the same keyboard.


Have now built upon this and created a version which allows the user to play against AI. Find it [here](https://github.com/dvtkdlwl/tic-tac-toe-ai).

---

## Features
- Local multiplayer (2 players)
- Turn-based gameplay
- Input validation
- Win detection
- Draw detection
- Runs entirely in the terminal

---

## How to Run

### Prerequisites
- Python 3.x installed

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/dvtkdlwl/1v1_TicTacToe.git

2. Navigate into the project folder:
    ```bash
    cd 1v1_TicTacToe.git

3. Run the game:
    ```bash
    python TTTGame.py


## How to Play

-> Player X goes first

-> Players alternate turns between X and O

-> Enter a number from 1 to 9 to place your mark on the board

-> Numbers correspond to positions on the Tic Tac Toe grid

-> The game ends when a player wins or when all spaces are filled (draw)
